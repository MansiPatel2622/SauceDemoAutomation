## SauceDemo Login Automation

## Objective

Automate the login functionality of SauceDemo using Selenium WebDriver.

## Test Cases Covered

1. Valid Login
2. Invalid Login
3. Empty Username
4. Empty Password

## Tools Used

* Java
* Selenium WebDriver
* Eclipse IDE
* Chrome Browser

## Setup Steps

1. Install Java JDK.
2. Install Eclipse IDE.
3. Download Selenium Java libraries.
4. Create Java Project.
5. Add Selenium JAR files to Build Path.
6. Create source code files.
7. Run LoginTest.java as Java Application.

## Expected Output

PASS : Valid Login
PASS : Invalid Login
PASS : Empty Username
PASS : Empty Password

## Report

Open reports/TestReport.html in a browser to view the test execution report.
